# USD and Turkish Inflation Analysis

This project analyzes the relationship between the Turkish CPI (inflation) and the USD/TRY exchange rate using OLS regression and visualization.

## Overview

- Reads USD and CPI (TÜFE) data from CSV and Excel files.
- Merges data based on dates.
- Performs a simple OLS regression to quantify the relationship.
- Visualizes results with scatter and line plots.
